Updating this drat

  1. Clone this repository if you don't have it already, and install the drat and crayon packages from CRAN
  2. If adding a new package, edit packages.txt to list github user name / package name. Blank lines and lines starting with "#" are ignored.
  3. Run ./build.R to download all packages, build tar balls, add to drat and make commits.
  4. Run git push to push the changes back here.
